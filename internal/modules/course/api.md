# Course API

Base URL: `/api/courses`

## Endpoints

### POST /courses
Create a new course
- Auth: Required

### GET /courses
Get all courses for current user
- Auth: Required

### GET /courses/active
Get only active courses
- Auth: Required

### GET /courses/{id}
Get course by ID
- Auth: Required

### PUT /courses/{id}
Update course
- Auth: Required

### DELETE /courses/{id}
Delete course
- Auth: Required

For complete API documentation, see `/api/openapi.yaml`
